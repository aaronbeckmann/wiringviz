# Keyboard & mouse reference

## Keyboard

| Key | Action |
| --- | --- |
| **V** | Move mode — drag components (default) |
| **W** | Wire mode — start wires from splice / terminal pads |
| **R** | Flip the selected connector (pins left↔right) or diode (A↔K) |
| **Delete** / **Backspace** | Delete the selection |
| **Shift + click / drag** | Multi-select |

## Mouse — canvas

| Gesture | Action |
| --- | --- |
| Drag empty space | Pan |
| Mouse wheel | Zoom |
| Drag from a pin | Draw a wire |
| Drag a component | Move it (drop on a board/group to attach) |
| Drag board corner (selected) | Resize board / group |

## Mouse — wires

| Gesture | Action |
| --- | --- |
| Click a wire | Select it |
| **Double-click a wire** | Add a layout point (bend) |
| Drag a bend diamond | Route the wire |
| **Double-click a bend** | Remove it |
| Drag a wire label | Move the label (leader arrow appears when far away) |
| **Double-click a label** | Reset the label position |
